# 量子隧穿效应的半经典推导

## WKB 近似

对于求解一维定态[[薛定谔方程]], 物理学家 [[Gregor Wentzel|Wentzel]], [[Hans Kramers|Kramers]] 以及 [[Léon Brillouin|Brillouin]] 发展了一种重要的半经典近似方法，常常称之为 [[WKB Approximation]]。这一近似仅在势能 $V(x)$ 变化得很缓慢时才成立。
