# Definition

>[!important] 定义
>归一化的球谐函数为
>$$Y_l^m(\theta, \phi) = \sqrt{\frac{2l+1}{4 \pi} \frac{(l-m)!}{(l+m)!}} P_l^m(\cos \theta) \ \text{e}^{\text{i} m \phi} \left\{\begin{aligned}
\ &l = 0, 1,2,\cdots\\
&m = 0, \pm 1, \pm2, \cdots 
\end{aligned}\right.$$
>
> 





# Property

## 正交归一性

From the [[连带勒让德函数#正交性|正交关系]] of the associated Legendre function
